    HORIZONS Ephemeris Examples

    The JPL HORIZONS platform provides ephemeris data as Cartesian state vectors, osculating orbital elements, and other representations. The precise REST API accepted by the JPL HORIZONS servers is implemented in Julia with HorizonsAPI.jl.

    julia> using HorizonsAPI
    julia> fetch_vectors(399; format="text") # Earth's position w.r.t. the solar system barycenterHTTP.Messages.Response:

    The response from the HORIZONS API contains a lot of information! To more simply fetch and inspect ephemeris data from JPL HORIZONS servers, use the ephemeris method in HorizonsEphemeris.jl.

    julia> using Dates, HorizonsEphemeris
    julia> ephemeris("earth", now())(t = [2.460427512474433e6], cal = InlineStrings.String31[" A.D. 2024-Apr-27 00:17:57.7910"], x = [-0.8121655506372288], y = [-0.55799542559173], z = [-0.2416542403861777], ẋ = [0.01005080185216747], ẏ = [-0.01268546221623465], ż = [-0.005498757459091392])

    The output of ephemeris is a NamedTuple, so you can easily pass it along to DataFrames.jl for convenient inspection & data processing. Note the wrt keyword argument as well; it is a simplified interface to the HORIZONS API CENTER argument.

    julia> data = DataFrame(
           ephemeris("earth", now() - Year(50), now() + Year(5), Day(1); wrt="jupiter")
